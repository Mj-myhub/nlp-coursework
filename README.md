# NLP Coursework

This repository brings together selected coursework, assignments, and notes from my M.Sc. in **Language and Mind: Linguistics and Cognitive Studies** at the **University of Siena**.

It reflects the academic side of my NLP background and covers a range of core topics, from text preprocessing and language modelling to transformers, embeddings, parsing, and computational psycholinguistics. I created this repo as a way to organise and present the work that helped shape my current interests in computational linguistics, language acquisition, and large language models.

## Topics Covered

| Module | Description |
|--------|-------------|
| Tokenisation & Preprocessing | Text normalisation, sentence splitting, and BPE |
| Language Modelling | N-gram models and perplexity evaluation |
| Word Embeddings | Word2Vec, GloVe, and contextual embeddings |
| Sequence Labelling | POS tagging and NER with BiLSTM-CRF |
| Syntax & Parsing | Constituency and dependency parsing |
| Transformers | Attention mechanisms and BERT fine-tuning |
| Computational Psycholinguistics | Surprisal theory and garden-path effects |

## Selected Areas of Work

### Text preprocessing and tokenisation
Exercises and notes related to text cleaning, tokenisation, segmentation, and subword methods.

### Language modelling
Coursework focused on probability estimation, smoothing, and evaluating language models through perplexity.

### Word embeddings
Work involving both static and contextual representations, with attention to how meaning is encoded across models.

### Sequence labelling
Assignments related to structured prediction tasks such as part-of-speech tagging and named entity recognition.

### Syntax and parsing
Coursework exploring linguistic structure through constituency and dependency-based approaches.

### Transformers and modern NLP
Exercises and readings connected to attention mechanisms, transformer architectures, and contextual language models.

### Computational psycholinguistics
Work linking NLP and language processing to topics such as surprisal, ambiguity, and sentence comprehension.

## Why this repository matters

This repo represents the academic foundation behind my later project work in NLP and LLMs. While my thesis and research repositories focus more on GPT-2, language acquisition, and psycholinguistically motivated evaluation, this one shows the broader training that supports that work.

It also reflects the way I like to approach NLP: not only as a technical field, but as something closely connected to linguistic theory, cognition, and real questions about how language is structured and processed.
