# NLP Coursework

Collection of NLP course assignments and exercises completed during M.Sc. Language and Mind at the University of Siena (2021).

## Topics Covered

| Module | Description |
|--------|-------------|
| Tokenisation & Preprocessing | Text normalisation, sentence splitting, BPE |
| Language Modelling | N-gram models, perplexity evaluation |
| Word Embeddings | Word2Vec, GloVe, contextual embeddings |
| Sequence Labelling | POS tagging, NER with BiLSTM-CRF |
| Syntax & Parsing | Constituency and dependency parsing |
| Transformers | Attention mechanisms, BERT fine-tuning |
| Computational Psycholinguistics | Surprisal theory, garden-path effects |

## Tools & Libraries

- Python 3.8+
- PyTorch, Hugging Face Transformers
- NLTK, spaCy
- Google Colab
